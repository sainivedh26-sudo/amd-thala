# Contributing to Thala

Thank you for your interest in contributing to [Project Name]! We welcome contributions from everyone, whether it's bug reports, feature requests, documentation improvements, or code enhancements.


## Code of Conduct

This project adheres to the [Contributor Covenant Code of Conduct](./CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code. Please report unacceptable behavior to [nivedh@cc.cc].

## Getting Started

### Fork and Clone

1. Fork the repository on GitHub
2. Clone your fork locally:
```
git clone https://github.com/SaiNivedh26/team-thala.git
cd team-thala
```

3. Add the upstream repository:
```
git remote add upstream https://github.com/SaiNivedh26/team-thala.git
```

### Types of Contributions

We welcome the following types of contributions:

- **Bug Reports**: Found a bug? Open an issue with detailed reproduction steps
- **Feature Requests**: Have an idea? Open an issue to discuss before implementing
- **Code Contributions**: Bug fixes, new features, or improvements
- **Documentation**: Improvements to README, guides, or inline code comments
- **Tests**: Additional test coverage is always appreciated
